# Home Tutor Web Application

A responsive web application developed using HTML, CSS, and JavaScript to connect students and tutors through a role-based platform.

## Features
- Login & Signup Authentication
- Student and Tutor Role Selection
- Tutor Dashboard
- KM Range Preference Selection
- Local Storage Session Handling
- Responsive UI Design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Future Improvements
- Backend Integration
- Database Support
- Live Tutor Search
- Authentication System
